O projeto "CORRIDALINHA" desenvolvido por Rafaela no editor p5.js é uma implementação interativa de um jogo de corrida simples, inspirado nos clássicos jogos de "navinha" dos anos 90. Utilizando a biblioteca p5.js, o jogo permite que o jogador controle uma nave espacial que deve desviar de obstáculos e coletar itens, proporcionando uma experiência dinâmica e envolvente.

Esse projeto é uma excelente demonstração de como é possível criar jogos interativos e divertidos utilizando ferramentas acessíveis como o p5.js, incentivando a criatividade e o aprendizado de programação.

O projeto "Quente ou Frio" desenvolvido por Rafaela no editor p5.js é uma implementação digital do clássico jogo de adivinhação, onde o jogador deve descobrir a posição de um número secreto com base em dicas de proximidade.

O objetivo do jogo é adivinhar um número aleatório gerado pelo sistema dentro de um intervalo específico. A cada tentativa, o jogo fornece feedback indicando se o palpite está "quente" (próximo) ou "frio" (distante) do número secreto.

O projeto "Monabinoqui" desenvolvido por Rafaela no editor p5.js é uma criação artística interativa que explora formas geométricas e cores vibrantes para proporcionar uma experiência visual dinâmica.

"Monabinoqui" é uma composição visual que utiliza elementos geométricos para criar padrões e animações que respondem ao movimento do mouse. A interação do usuário influencia diretamente a disposição e a aparência dos elementos na tela, oferecendo uma experiência única a cada interação.
